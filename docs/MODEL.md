# モデルの考え方

## 目的

実物の材料定数を再現することではなく、次の因果関係を中学生が操作しながら理解できるようにすることです。

1. 指が進む
2. フィルムに変形エネルギーがたまる
3. 引裂抵抗に相当するしきい値を超える
4. 破れの先端が少し進む
5. 先端の新しい位置が、次の進み方に影響する

## 波状軌跡

現在のモデルでは、中心線からのずれに応じて反対側へ戻す働きを加えています。
ただし向きは瞬時には切り替わらず、速度に相当する状態量が残るため、先端が中心を行き過ぎます。
この「戻す働き」と「行き過ぎ」の繰り返しを、減衰のある振動として単純化しています。

## 実物との差

実物では、少なくとも次の影響が考えられます。

- 指とフィルムの摩擦
- 指の前後での膜の曲げ、しわ、たわみ
- 面内引張りとせん断の混合
- 膜厚と材料特性のばらつき
- 切り込み形状
- 破断速度への依存
- 材料の塑性変形や粘弾性

したがって、このモデルだけから実物の破断機構を断定してはいけません。
